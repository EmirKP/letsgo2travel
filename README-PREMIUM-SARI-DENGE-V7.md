Bu paket, sarı rengi #FFDF00 olarak sabitler ancak siteyi sarıya boğmadan premium kullanım dengesine çeker.

- #FFDF00 sadece CTA, logo vurgusu, aktif çizgi ve küçük marka detaylarında tutuldu.
- Link/bilgi rengi sarıdan tekrar okyanus teal tonuna alındı.

- Hero üst rozeti cam kapsül görünümünden çıkarılıp küçük premium eyebrow stiline çekildi.
