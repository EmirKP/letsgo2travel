# LetsGo2Travel Cinematic Palette v11

Bu paket LetsGo2Travel renk paletini ve görsel hiyerarşisini sinematik reel + profesyonel lokasyon danışmanlığı vizyonuna göre günceller.

## Uygulanan palet

- Ana arka plan: `#F8F9FA`
- Header/Footer ve koyu bantlar: `#0F172A`
- CTA ve vurgu: `#FF6B35`
- CTA hover: `#E85D04`
- Paragraf: `#334155`
- Kart arka planı: `#FFFFFF`

## Tasarım etkisi

- Sarı marka denemesi kaldırılıp coral/turuncu sinematik CTA çizgisine geçildi.
- Header/footer koyu lacivert sabitlendi.
- Kartlar beyaz + soft shadow yapıldı.
- Video/reel kartları için 12px radius ve hover scale 1.02 eklendi.
- Başlık fontu Playfair Display, metin fontu Montserrat oldu.
- Genel transition süresi 0.3s ease-in-out olarak sabitlendi.
- Hero daha sinematik koyu lacivert + coral vurgu yapısına çekildi.
