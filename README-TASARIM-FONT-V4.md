# LetsGo2Travel Tasarım + Font V4

Bu paket, V3'te gelen `Manrope + Plus Jakarta Sans` font sistemini kaldırır.

## Yeni yaklaşım

- Google font yok.
- Sistem fontu var: Segoe UI / Apple system / Roboto fallback.
- Başlıklar daha hafif ve premium: `font-weight: 700`.
- Butonlar ve nav daha sakin: aşırı kalın oyuncak hissi azaltıldı.
- Hero başlığı küçültüldü, dev afiş hissi azaltıldı.
- Dropdown yazıları net görünür hale getirildi.
- Search tabları ve AI alanı okunur tutuldu.
