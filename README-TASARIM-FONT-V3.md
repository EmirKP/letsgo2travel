# LetsGo2Travel Tasarım + Font V3

Bu paket, canlı ekranda görülen tasarım problemlerini düzeltmek için hazırlandı.

## Düzeltilenler

- Site fontu `Manrope + Plus Jakarta Sans` sistemine alındı.
- Header daha sıkışık olmayan, premium navigasyon yapısına çekildi.
- `Daha Fazla` dropdown menüsünde sadece ikon görünmesi sorunu düzeltildi.
- Uçak/Otel arama tablarında pasif sekme okunur hale getirildi.
- AI Planlayıcı sayfasında beyaz zemin üzerinde beyaz yazı problemi düzeltildi.
- Ana renk dili daha seyahat sitesine uygun hale getirildi:
  - Gece laciverti: güven ve marka zemini
  - Okyanus/teal: link ve bilgi vurgusu
  - Gün batımı gold: CTA ve aksiyon rengi
  - Açık gökyüzü/kum beyazı: ana sayfa ferahlığı
- Kartlar ve paneller daha temiz, beyaz, okunur hale getirildi.
- Aşırı hover/glow efekti yumuşatıldı.
