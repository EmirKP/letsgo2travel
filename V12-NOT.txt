Letsgo 2 Travel V12 büyük tasarım güncellemesi

Bu sürümde yapılanlar:
- Ana sayfa yeniden tasarlandı: premium hero, gelişmiş arama alanı, canlı fırsat vitrini, kategori vitrinleri, fiyat alarmı CTA ve güven açıklamaları eklendi.
- Admin panel sıfırdan profesyonel yönetim merkezi görünümüne taşındı.
- Admin panelde sol menü, istatistik kartları, kalite kontrol, sosyal medya metni, canlı önizleme, görsel URL kontrolü ve CSV indir alanları geliştirildi.
- Fırsatlar sayfası /flights katalog mantığına çevrildi; kategori, vize, sıralama ve arama filtreleri eklendi.
- Arama sayfası daha profesyonel sonuç ekranı haline getirildi; filtre paneli, sonuç kartları ve fiyat alarmı alanı geliştirildi.
- Yeni public API eklendi: /api/firsatlar. Bu API aktif tüm fırsatları kullanıcı tarafına güvenli şekilde çeker.
- Görsel URL eklenen rotalar ana sayfa, flights ve arama sayfasında görünür.
- Build testi dummy environment ile başarıyla geçti.

Deploy komutları:
git add .
git commit -m "v12 buyuk tasarim ve admin gelistirmesi"
git push origin main

Not:
.env.local dosyanı silme. Zip içine güvenlik sebebiyle env dosyası konmadı.
